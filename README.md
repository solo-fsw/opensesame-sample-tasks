# OpenSesame Sample Tasks

This repo contains OpenSesame sample tasks created/modified by SOLO.

To download this repo (and all samples in it), click on the green Code button and click Download ZIP.

This repo contains the sample tasks below. See each folder for more information.

* **pygaze**: A sample task uses the PyGaze eye tracking library. See the folder for more info.
* **titta package**: A sample task uses the titta package to record data with Tobii Pro Lab (External Presenter Mode). See the folder for more info.
